┌─────────────────────────────────────────────────────────────────────┐
│ ⚠️  TROUBLESHOOTING                                                 │
└─────────────────────────────────────────────────────────────────────┘

  CORS Error:
  → Add your URL to allow_origins in api.py
  
  Port in Use:
  → lsof -ti:8000 | xargs kill -9
  
  Module Not Found:
  → pip install -r api_requirements.txt
  
  Can't Connect to API:
  → Make sure API is running: python api.py
