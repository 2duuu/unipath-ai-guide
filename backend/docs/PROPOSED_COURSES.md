## Implementation Notes

1. **Verification Required**: All courses listed must be verified against official university curriculum pages
2. **Course Count**: 
   - Bachelor programs: 5-6 courses (representative sample)
   - Master programs: 4-5 courses (specialized topics)
   - 6-year medical programs: 7-9 courses (comprehensive curriculum)
3. **Uniqueness**: Courses are unique per program, but same course names can appear in different programs (e.g., "Machine Learning" can be in both Computer Science and AI programs)
4. **Metadata**: When implementing, courses should include:
   - Semester (which semester the course is typically offered)
   - Credits (ECTS credits)
   - Core vs Elective status
   - Prerequisites (if any)
5. **Appeal Factor**: Courses are named to be attractive and specific, using terms like "Advanced", "Modern", specific technologies (IoT, AI, etc.) to attract student interest
